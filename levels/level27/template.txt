WELCOME TO LEVEL 27 — TERMINAL INSIDE VIM
=========================================

🏆  Mission  
Open a terminal window inside Vim and reveal the magic word.

────────────────────────────────────────
📝  Step-by-step

1. From Normal mode, type `:terminal` to open a terminal buffer.
2. Run the command: `echo "The magic word is abracadabra"`
3. Press `Ctrl-w n` to return to Normal mode from the terminal.
4. Type `:q` to close the terminal window.
5. Save and quit with `:wq`.

Tip: You can open and interact with real shells from inside Vim!
