WELCOME TO LEVEL 36 — :WINDO COMMAND
====================================

🏆  Mission  
Use the `:windo` command to apply a setting across all split windows.

────────────────────────────────────────
📝  Step-by-step

1. Split the current file horizontally using:

   :split

2. Run the following command to enable line numbers in both:

   :windo set number

3. Confirm that line numbers appear in both windows
4. Save and quit with:

   :wq
