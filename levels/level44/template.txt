WELCOME TO LEVEL 44 — TURN TODOs INTO CHECKBOXES
================================================

🏆 Mission  
Convert every line that starts with  `TODO:`  into a markdown checkbox  
`- [ ] …`  using **one** substitute command.

────────────────────────────────────────
📝 Step-by-step

1. Make sure you’re in Normal mode.

2. Run this single command (one line):

        :%s/^TODO: /- [ ] /g

    • `^TODO: `   matches only lines that begin with “TODO: ”  
    • `- [ ] `    is the replacement checkbox prefix  
    • `g`         applies it to every match in the buffer.

3. Save and quit:   `:wq`
────────────────────────────────────────
✅ Success checklist  
    • No line starts with `TODO:`  
    • Every former TODO line now starts with `- [ ] `
────────────────────────────────────────
