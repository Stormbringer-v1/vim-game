WELCOME TO LEVEL 28 — EDITING .vimrc
====================================

🏆  Mission  
Make a change in your `.vimrc` and reload it from inside Vim.

────────────────────────────────────────
📝  Step-by-step

1. Open your Vim config: `:e ~/.vimrc`
2. Add this line at the end:
   `set number`
3. Save the file with `:w`
4. Reload it with `:source %`
5. Return to this buffer with `:e #` and `:wq` to quit

Tip: `%` is a shortcut for the current file in commands like `:source %`.

────────────────────────────────────────
✅  Success checklist  
- [ ] `.vimrc` was edited and saved
- [ ] The config was reloaded inside Vim
- [ ] You returned and exited this buffer
