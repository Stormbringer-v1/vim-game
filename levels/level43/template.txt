WELCOME TO LEVEL 43 — BULLET POINT MAGIC
========================================

🏆  Mission  
Turn each line into a proper bullet-point list using Visual Block mode or macros.

────────────────────────────────────────
📝  Step-by-step

1. Visually select all task lines (use `Ctrl+v`, then move down)
2. Insert `- ` at the start of each line with:

   I- <Esc>

3. Your list should look like:

   - Buy milk  
   - Arrange meeting  
   - Write blog post  
   - Push code to GitHub  
   - Walk the dog

4. Save and quit with `:wq`

────────────────────────────────────────
✅  Success checklist  
- [ ] All tasks prefixed with `- `  
- [ ] File looks like a clean bullet list  
- [ ] File saved and exited properly
