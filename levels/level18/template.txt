WELCOME TO LEVEL 18 — ADDING TODOs
==================================

🏆  Mission  
Add `[TODO]` at the beginning of every line in the block.

────────────────────────────────────────
📝  Step-by-step

1. Use `Ctrl-v` to enter Visual Block mode.
2. Select the first column of all lines.
3. Press `I`, type `[TODO]`, then press `Esc`.
4. Watch Vim insert the text on every selected line!
5. Save and quit with `:wq`.

Tip: Visual block insert with `I` is a Vim superpower.

────────────────────────────────────────
✅  Success checklist  
- [ ] Every line starts with `[TODO]`
- [ ] You used block insert (not manual edits)
- [ ] You saved and exited properly
