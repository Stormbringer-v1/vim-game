WELCOME TO LEVEL 37 — SORTING LINES
===================================

🏆  Mission  
Sort a list of words in alphabetical order using Vim’s built-in sort.

────────────────────────────────────────
📝  Step-by-step

1. Select all the lines inside the task block using visual mode:  
   Press `ggVG`

2. Run the sort command:  
   `:sort`

3. Confirm that the fruit names are now in alphabetical order

4. Save and quit with:  
   `:wq`

────────────────────────────────────────
✅  Success checklist  
- [ ] Lines are sorted alphabetically (A to Z)  
- [ ] No lines were lost or duplicated  
- [ ] File was saved and exited correctly
