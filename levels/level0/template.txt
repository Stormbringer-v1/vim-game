WELCOME TO LEVEL 0 — HOW TO QUIT VIM
====================================

🏆  Mission  
Learn how to quit Vim using the proper command.

────────────────────────────────────────
📝  Step-by-step

1. Look at the text below. You don’t need to change anything.
2. Press `:` (colon) to enter command mode.
3. Type `wq` — short for “write and quit.”
4. Press `Enter` to exit Vim safely.

If you do it right, you'll pass this level.

────────────────────────────────────────
✅  Success checklist  
- [ ] You didn’t panic
- [ ] You typed `:wq` and hit Enter
- [ ] You exited Vim and completed the level
