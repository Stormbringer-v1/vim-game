WELCOME TO LEVEL 11 — VISUAL BLOCK MODE
=======================================

🏆  Mission  
Use Visual Block mode to remove all the `X`s at the end of the lines — in one move.

────────────────────────────────────────
📝  Step-by-step

1. Move your cursor to the first `X`.
2. Press `Ctrl-v` to enter Visual Block mode.
3. Use `j` to move down and select all `X`s.
4. Press `x` or `d` to delete the selected column.
5. Save and quit with `:wq`.

Tip: Visual Block mode lets you edit columns of text — it’s incredibly powerful!

────────────────────────────────────────
✅  Success checklist  
- [ ] All `X`s are gone (cleanly!)
- [ ] You used Visual Block mode with `Ctrl-v`
- [ ] You saved and exited properly
